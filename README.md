## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/estlosan/solidity-react-aave
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run WebAPP
    ```sh
    cd client & npm run start
    ```
